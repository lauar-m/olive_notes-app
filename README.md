# olive notes

Um aplicativo web de produtividade inspirado em post-its, com foco em organização visual e planejamento pessoal.

## ✨ Funcionalidades

- Criar e gerenciar tarefas (dia, semana, mês)
- Drag and drop entre categorias
- Sistema de metas (anuais e de vida)
- Autenticação de usuários
- Interface inspirada em mid-century modern

## 🛠️ Tecnologias

- JavaScript
- Node.js + Express
- MongoDB
- Bootstrap
- Font Awesome
